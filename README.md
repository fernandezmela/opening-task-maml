# MAML on Abstract Strategy Games

This project investigates whether meta-learning can accelerate learning across board games. Using Model-Agnostic Meta-Learning (MAML), neural networks are trained on Chess tasks and adapted to Shogi, allowing the model to leverage shared strategic representations while requiring fewer task-specific updates. The framework combines supervised pretraining on large-scale game datasets with reinforcement learning in game environments to evaluate cross-domain transfer performance.
